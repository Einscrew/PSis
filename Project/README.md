# Project

### Build
`Psis/Project: $ make`

Detailed options are:

`Psis/Project: $ make app`

`Psis/Project: $ make clipboard`

### Run
On single mode:

`Psis/Project/cliboard: $ ./clipboard `

On connected mode:

`Psis/Project/cliboard: $ ./clipboard -c {IP} {PORT}`

### Clean
`Psis/Project: $ make clean`
